# Medium Clone FullStack

## Teach stack

    - NodeJS
    - GraphQl
    - ReactJS
    - MongoDB
    - Apollo

### .env Setup

Create a .env file in root directory and assign all these field.
Currently We are using MongoDB Atlas

```
SECRET_KEY = SOMESUPERSECRETKEY
DB_PASSWORD = <Your MongoDB Database Password>
DB_USERNAME = <Your MongoDB Database User Name>
APP_PORT = 8080
```

## How to run project

```
    npm install
    npm run dev
```
