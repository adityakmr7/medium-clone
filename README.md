# Medium Clone FullStack

## Teach stack

    - NodeJS
    - GraphQl
    - ReactJS
    - MongoDB
    - Apollo

## How to run project

    ##### Client only

    ```
    npm run dev:client
    ```

    ##### Server only

    ```
    npm run dev:server
    ```

    ##### Running both

    ```
    npm run dev:dev
    ```
